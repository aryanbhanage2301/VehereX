# VEHEREX---A-Platform-To-Find-Your-Perfect-Exotic-Ride-
VEHEREX is a web-based personalized Electric Vehicle (EV) recommendation platform designed to address India’s low EV adoption caused by range anxiety, limited charging infrastructure, and decision complexity.

The platform simplifies EV decision-making by recommending suitable electric vehicles based on:
User budget
Usage patterns
Charging availability
It supports sustainable mobility aligned with the United Nations Sustainable Development Goals (SDGs).
